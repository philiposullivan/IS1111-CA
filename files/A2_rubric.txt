## QUICK REFERENCE: What each function must do

| Function | Key Requirements |
|----------|------------------|
| `generate_product_id(inventory)` | Return "P001" if empty; otherwise find highest existing ID number, add 1, format as "P" + 3 zero-padded digits |
| `search_products(inventory)` | Ask user for (N)ame or (C)ategory; name search = partial match, case-insensitive; category search = exact match, case-insensitive; display formatted results with count |
| `log_transaction(transactions, trans_type, product_id, product_name, quantity)` | Create transaction dict with type, product_id, product_name, quantity, timestamp (ISO format); append to transactions list |
| `view_transaction_log(transactions, num_recent=10)` | Display most recent transactions (newest first); format timestamp as "YYYY-MM-DD HH:MM:SS"; show quantity with +/- sign |
| `generate_category_report(inventory)` | Group products by category; count products and sum value (price Ã— quantity) per category; display formatted report with totals; return the category data dict |
